# Frogger-GPS

### Instrucciones:

#### 1. Descargar el repositorio con git clone:

```
git init
git clone https://github.com/GonzaVice/GPS-Frogger.git
```

#### 2. Crear una virtual enviroment:

```
python3 -m venv .venv
```

#### 3. Activar virtual enviroment:

En Windows:

```
.\.venv\Scripts\Activate
```

En MacOS/Linux:

```
source .venv/bin/activate
```
